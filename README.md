# موقع صيدلية الشعب

موقع ويب احترافي كامل مبني بلغة بايثون (Flask) لصيدلية "الشعب" تحت إشراف د. محمد حسن.

## المحتويات
- الصفحة الرئيسية (`/`)
- صفحة عن الصيدلية (`/about`)
- صفحة الخدمات (`/services`)
- صفحة التواصل مع نموذج فعلي (`/contact`)

## طريقة التشغيل

1. تأكد إن بايثون 3.9+ مثبت عندك.
2. افتح Terminal في مجلد المشروع وثبّت المكتبات:

```bash
pip install -r requirements.txt
```

3. شغّل السيرفر:

```bash
python app.py
```

4. افتح المتصفح على:

```
http://127.0.0.1:5000
```

## التعديل على بيانات الصيدلية

كل بيانات الصيدلية (الاسم، رقم التليفون، العنوان، المواعيد، الخدمات، آراء العملاء)
موجودة في أعلى ملف `app.py` داخل القواميس التالية، وتقدر تعدّلها زي ما تحب من غير
ما تلمس باقي الكود:

- `PHARMACY` — البيانات الأساسية (الاسم، التليفون، العنوان، المواعيد...)
- `SERVICES` — قائمة الخدمات المعروضة
- `TESTIMONIALS` — آراء العملاء

## هيكل المشروع

```
alshaab_pharmacy/
├── app.py                  # تطبيق Flask الرئيسي (الراوتس والبيانات)
├── requirements.txt
├── templates/
│   ├── base.html            # القالب الأساسي (الهيدر والفوتر)
│   ├── index.html           # الصفحة الرئيسية
│   ├── about.html           # عن الصيدلية
│   ├── services.html        # الخدمات
│   └── contact.html         # التواصل
└── static/
    ├── css/style.css        # التنسيقات (هوية بصرية كاملة)
    └── js/main.js           # قائمة الموبايل
```

## ملاحظات مهمة قبل النشر الحقيقي

- غيّر قيمة `app.secret_key` في `app.py` لقيمة سرية عشوائية.
- نموذج التواصل حاليًا بيطبع الرسالة في الـ console فقط؛ لو عايز تستقبلها
  فعليًا، تقدر تربطه بإيميل (smtplib) أو قاعدة بيانات بسيطة (SQLite).
- استبدل رقم الواتساب والتليفون والعنوان بالبيانات الحقيقية في `PHARMACY`.
- الخريطة المدمجة حاليًا عامة على بورسعيد؛ استبدلها برابط Google Maps
  الفعلي لموقع الصيدلية.
